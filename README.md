# Inventory Checker


##  Basic Description
Takes path to item on webpage and checks if it's currently in stock


## To run
- Go to project repo
- Create files mentioned in index.js and sendEmail.js and fill with necessary data
- Run using the following command in the terminal
  - node index.js
